# 音乐汇 - Music Festival

一个基于 Vite + React 的现代化音乐播放器应用,支持多平台音乐搜索、播放、收藏、下载等功能。

## 功能特性

### 核心功能
- ✅ **多源音乐搜索** - 支持网易云、QQ音乐、酷狗、酷我等13个音乐平台
- ✅ **在线播放** - 流畅的音乐播放体验,支持多种音质选择(128K-Hi-Res)
- ✅ **歌词显示** - 实时滚动歌词,支持点击跳转
- ✅ **网易云歌单解析** - 输入歌单ID快速导入整个歌单

### 新增功能
- ✨ **收藏单曲** - 将喜欢的歌曲添加到收藏,基于 localStorage 持久化存储
- ✨ **收藏列表** - 创建自定义歌单,管理你的音乐收藏
- ✨ **单曲下载** - 支持下载音乐和歌词文件
- ✨ **批量下载** - 全选或选择多首歌曲批量下载
- ✨ **随机推荐** - 一键获取随机推荐音乐并播放
- ✨ **最近播放** - 自动记录播放历史,快速回顾

### 用户体验
- 📱 **响应式设计** - 完美适配PC端和移动端
- 🎨 **精美UI** - 渐变背景动画、毛玻璃效果、流畅过渡动画
- 🎵 **旋转封面** - 播放时专辑封面旋转动画
- 💾 **数据持久化** - 所有收藏和设置自动保存到本地
- ⌨️ **键盘快捷键** - 空格播放/暂停,左右箭头切歌

## 技术栈

- **前端框架**: React 18
- **构建工具**: Vite
- **状态管理**: Context API + Hooks
- **样式**: CSS3 (Flexbox + Grid)
- **图标**: Font Awesome 6
- **API**: music-api.gdstudio.xyz

## 安装运行

1. **安装依赖**
```bash
cd music-festival-react
npm install
```

2. **启动开发服务器**
```bash
npm run dev
```

3. **访问应用**
打开浏览器访问: http://localhost:5173

4. **构建生产版本**
```bash
npm run build
```

## 使用说明

### 搜索音乐
1. 在顶部搜索框输入歌曲名、歌手或专辑
2. 选择音乐平台(默认网易云)
3. 点击搜索或按回车键

### 播放音乐
- 点击歌曲列表中的任意歌曲开始播放
- 使用播放器控制播放/暂停、上一曲/下一曲
- 拖动进度条跳转播放位置
- 调节音量滑块控制音量大小

### 收藏功能
- 点击歌曲右侧的 ❤️ 图标收藏单曲
- 点击顶部导航的 ❤️ 按钮查看收藏列表
- 点击 ➕ 图标添加歌曲到自定义歌单
- 在收藏列表页面创建、管理自定义歌单

### 下载功能
- **单曲下载**: 点击歌曲右侧下载按钮
- **批量下载**: 勾选多首歌曲后点击"批量下载"
- **下载歌词**: 点击歌词下载按钮

### 其他功能
- **网易云歌单**: 输入歌单ID解析整个歌单
- **随机推荐**: 点击"随机推荐"获取推荐音乐
- **最近播放**: 查看播放历史记录

## 数据存储

所有数据使用 localStorage 存储,包括:
- 收藏的单曲
- 自定义歌单
- 最近播放记录
- 音质和音量设置

## 响应式断点

- **大屏幕** (>1400px): 三栏布局(搜索+播放器+歌词)
- **中等屏幕** (1024px-1400px): 两栏布局(搜索+播放器)
- **移动端** (<1024px): 单栏布局,垂直排列

## 浏览器支持

- Chrome (推荐)
- Firefox
- Edge
- Safari
- 移动端浏览器

## 注意事项

1. 音乐API依赖第三方服务,可能存在不稳定情况
2. 部分平台可能需要VIP或有地区限制
3. 下载功能依赖浏览器下载管理器
4. 建议在良好的网络环境下使用
